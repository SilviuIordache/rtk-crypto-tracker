# RTK Crypto Tracker

React + Redux Toolkit Query demo that shows live BTC/ETH/SOL prices.

## Run
- `npm install`
- `npm run dev`

## API notes
- The app uses Coinranking through Vite dev proxy (`/api`).
- Optional: set `COINRANKING_API_KEY` in your shell before running dev server to use your key:
  - `export COINRANKING_API_KEY=your_key_here`

If the dev server is already running, restart it after config changes.
